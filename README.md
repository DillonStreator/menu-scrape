# menu-scrape

Scrape menus from websites


## Support

- [x] Grubhub
- [ ] Door Dash
- [ ] Uber Eats

## Use

```sh
# clone 
git clone https://github.com/DillonStreator/menu-scrape

# move into project
cd menu-scrape

# install dependencies
npm install

# run program
node menu_scrape.js https://www.grubhub.com/restaurant/dan-thai-2251-w-schaumburg-rd-schaumburg/833809
```
